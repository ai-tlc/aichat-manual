- **Prompt** — Your instruction, question, or text for the AI. The foundation of everything.
- **Token** — The smallest text unit for the AI. The **context window** is measured in tokens.
- **Context Window** — The model’s “working memory”: maximum tokens processed at one time.
- **Persona** — A set of instructions that gives the AI a specific role or style (e.g., expert, coach).
- **Hallucination** — When the AI presents convincing but incorrect information. Always double‑check.
- **Prompt Engineering** — Writing effective prompts to get exactly the output you want.
- **API** — An interface allowing software to communicate. **Extensions** use this.
- **Extension** — An extra tool/plugin that gives UvA AI Chat new features (often via external systems).
- **Bias** — Unintended skew from training data leading to unfair answers.
- **LLM** — A large AI model trained on vast text to understand and generate human language.
